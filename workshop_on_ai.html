3-Day Intensive Workshop for Researchers and Practitioners

Author

Prof. Dr. Zahid Asghar

Published

November 4, 2025

Workshop Overview

Target Audience & Prerequisites

Participants: Advanced researchers, policy analysts, business intelligence professionals, and doctoral students in economics, business, and social sciences.

Prerequisites: - Intermediate knowledge of econometrics and statistical modeling - Proficiency in R or Python for data analysis - Experience with research design and methodology - Basic understanding of machine learning concepts - Familiarity with academic writing and publication processes

Difficulty Level: Above Average (70-80th percentile)

Learning Architecture

This workshop employs a problem-based learning approach where participants tackle real-world economic and social challenges using cutting-edge AI methodologies. Each session integrates theoretical frameworks with hands-on implementation using R, Python, and AI-assisted research workflows.

Day 1: Advanced AI-Driven Research Design

Learning Objectives

Upon completion, participants will:

  1. Design sophisticated research frameworks using AI for complex economic phenomena
  2. Implement advanced prompt engineering for academic and policy research
  3. Evaluate AI-generated hypotheses using economic theory and empirical validation
  4. Construct reproducible research pipelines integrating AI tools with traditional econometric methods

Session Structure (2 hours)

Theoretical Foundation

  3. Behavioral Economics: Social media influence on consumer decision-making

Applied Exercise: Trade and Development

Research Question: How do bilateral trade agreements affect income inequality within developing countries, and how do these effects vary by institutional quality and industrial structure?

Code 
library(fixest)
library(did)
library(grf)
library(modelsummary)

# AI-assisted model specification prompt
trade_model_prompt <- "
RESEARCH CONTEXT:
- Dependent variable: Gini coefficient (income inequality)
- Treatment: Bilateral trade agreement implementation
- Unit of analysis: Country-year (1990-2020)
- Sample: 85 developing countries

THEORETICAL FRAMEWORK:
- Stolper-Samuelson theorem predictions
- Heckscher-Ohlin model extensions
- Institutional economics (North, 1990)

ECONOMETRIC CHALLENGES:
1. Endogenous treatment (trade agreements not randomly assigned)
2. Time-varying confounders (economic reforms, political changes)
3. Heterogeneous treatment effects by:
   - Institutional quality (governance indicators)
   - Industrial structure (manufacturing vs. primary exports)
   - Initial inequality levels

SPECIFICATION REQUIREMENTS:
- Propose instrumental variable strategy
- Design difference-in-differences with multiple periods
- Include heterogeneous effects analysis
- Address potential spillover effects

Provide complete R code with robustness checks.
"

# Econometric implementation
trade_model <- feols(
  gini ~ i(post_agreement, institutional_quality, ref = 0) + 
         manufacturing_share + gdp_pc + population |
         country + year,
  data = trade_data,
  cluster = ~country
)

# Heterogeneous effects using causal forests
cf_model <- causal_forest(
  X = trade_data[, c("institutional_quality", "manufacturing_share", "gdp_pc")],
  Y = trade_data$gini,
  W = trade_data$trade_agreement,
  num.trees = 4000
)

4. Educational outcomes for children

DATA STRUCTURE:
- Unit: Census tract-year
- Treatment: Affordable housing units per capita
- Outcomes: Multiple (property values, demographics, etc.)
- Confounders: High-dimensional (200+ variables)

IDENTIFICATION STRATEGY:
Use double machine learning with:
- Random Forest for nuisance function estimation
- Causal Forest for heterogeneous effects
- Cross-fitting for bias reduction

Consider heterogeneity by:
- Baseline neighborhood characteristics
- City size and housing market conditions
- Policy implementation details

Provide interpretation framework for policy recommendations.
"""

# Implementation using EconML

    model_y=RandomForestRegressor(n_estimators=100),
