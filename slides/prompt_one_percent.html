The Three Pillars of Prompt Thinking

1️⃣ First Principles Thinking

Break complex problems into core elements

Process Flow: - Complex Problem → Break Down → Core Elements → Rebuild with Clarity

  • Question everything, assume nothing

  • Build from fundamentals

2️⃣ Chain of Thought

Layer reasoning step by step

Process Flow: - Layer 1: Context → Layer 2: Analysis → Layer 3: Synthesis → Layer 4: Action

  • Sequential reasoning

  • Transparent thinking

3️⃣ Meta-Prompting

Think about thinking itself

Process Flow: - Think about Thinking → Design the Process → Build the System → Scale the Solution

  • System-level optimization

  • Self-improving processes

The 6 Prompt Atoms ⚛️

🎯 Goal State

The specific transformation you want to achieve

Example: “Transform raw survey data into actionable insights with 3 strategic recommendations”

📦 Source Material

What data needs to be preserved or transformed

Example: “Using the attached customer feedback CSV with 500 responses”

🚧 Constraints

Boundaries and limitations

Example: “Maximum 500 words, executive tone, compliant with GDPR”

🔄 Process

Step-by-step thinking instructions

Example: “First categorize by sentiment, then identify patterns, finally prioritize by impact”

✅ Validation

Success criteria and examples

Example: “Output should match this format: [template], include confidence scores”

🔁 Iteration

How feedback should be handled

Example: “If unclear, ask for clarification on business priorities first”

Real Example: Job Description 💼

❌ Basic Prompt

"Write a job description 
for an accountant"

Result: Generic, forgettable, no differentiation

✅ First Principles Prompt

"Write a job description for an accountant 
joining a fast-moving media company where 
AI is heavily integrated.

Role includes:
- Outcome ownership over financial tracking
- Automation oversight
- Cash flow forecasting

Tone: Human, for proactive professionals 
wanting to grow with a lean, intelligent team

Include: 3 unique differentiators reflecting 
our culture of innovation and transparency"

Result: Magnetic, filtering, aligned

Chain of Thought in Action 🔗

Building an Onboarding Sequence

1️⃣ “What are the top 3 emotions a new client feels in week one?”
→ Uncertainty, excitement, overwhelm

2️⃣ “How can we shift those into confidence and clarity?”
→ Quick wins, clear roadmap, personal touch

3️⃣ “Write the first email to do exactly that—short, empathetic”
→ Personalized welcome with immediate value

4️⃣ “Turn this into a 1-minute voice note script, friendly founder tone”
→ Authentic connection at scale

5️⃣ “What automation would increase response rate by 30%?”
→ Behavioral triggers + personalization

The Professional Framework 🏗️

T - Task

# Clear, Specific Instruction
"Analyze customer churn data and identify 
the top 3 predictive factors"

C - Context

# Background Information
"B2B SaaS company, $10M ARR, 
500 enterprise clients, 
18-month average contract"

R - References

# Examples & Style Guides
"Similar to McKinsey's issue trees, 
use MECE principle, 
reference our brand voice guide"

E - Evaluate

# Success Criteria
"Accuracy > 85%, 
actionable recommendations, 
C-suite ready presentation"

I - Iterate

# Feedback Loops
"First draft for data validation, 
second for strategic alignment, 
final for executive review"
